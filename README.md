# Shiftly PWA（プレゼン用）

## GitHubへ保存する手順

1. GitHubで新しいリポジトリを作成します。
2. このフォルダ内のファイルと `icons` フォルダを、構成を変えずにすべてアップロードします。
3. リポジトリの **Settings → Pages** を開きます。
4. **Deploy from a branch**、`main`、`/(root)` を選んで保存します。
5. 数分後に表示されるGitHub PagesのURLをChromeまたはEdgeで開きます。
6. ブラウザの「アプリをインストール」からPCへ追加します。

## 更新方法

GitHub上のファイルを更新すると、各デバイスは次回オンライン起動時に最新版を確認します。新しい版が見つかると自動的に切り替わり、画面を再読み込みします。オフライン時は最後に取得した版が起動します。

更新内容を公開するときは、画面上部の「最終更新日」と `sw.js` のキャッシュ名も変更してください。`index.html`、`app.js`、`styles.css`、`sw.js`、`manifest.webmanifest`、`icons` を含むフォルダ一式を上書きします。

このPWAは2026年7月17日時点の非公開Sites版と同じ機能構成を保つ、会社プレゼン用正規版です。休日区分の即時反映、動的アラート、特休申請、有給管理、残業登録、部所長・総務画面、スマホ表示を含みます。簡略版は使用しません。

## 注意

- プレゼン用プロトタイプです。入力内容はブラウザ内だけに保存され、他のPCとは共有されません。
- 実在社員の個人情報は入力しないでください。
- PWAはHTTPSで公開する必要があるため、ファイルを直接ダブルクリックするだけではインストールできません。GitHub Pages上で確認してください。
